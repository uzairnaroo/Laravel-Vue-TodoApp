Introduction

Task Manager API Documentation

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_API_TOKEN}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

Endpoints

Register a new user.

Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"John Doe\",
    \"email\": \"john.doe@example.com\",
    \"password\": \"password123\",
    \"password_confirmation\": \"password123\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "John Doe",
    "email": "john.doe@example.com",
    "password": "password123",
    "password_confirmation": "password123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (201):


{
    "access_token": "token123",
    "token_type": "Bearer",
    "user": {
        "id": 1,
        "name": "John Doe",
        "email": "john.doe@example.com"
    }
}

    --get "http://localhost/api/tasks?is_completed=1&priority=High&search=Task" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/tasks"
);

const params = {
    "is_completed": "1",
    "priority": "High",

};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

Store a new task.

Create a new task for the authenticated user.

Example request:
curl --request POST \
    "http://localhost/api/tasks" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "title=New Task"\
    --form "description=Task description"\
    --form "due_date=2025-01-23"\
    --form "priority=High"\
    --form "tags[]=inqbb"\
    --form "attachments[]=@/tmp/phpvRwHly" 
const url = new URL(
    "http://localhost/api/tasks"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('title', 'New Task');
body.append('description', 'Task description');
body.append('due_date', '2025-01-23');
body.append('priority', 'High');
body.append('tags[]', 'inqbb');
body.append('attachments[]', document.querySelector('input[name="attachments[]"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());

Example response (201):


{
    "id": 1,
    "title": "New Task",
    "description": "Task description",
    "priority": "High",
    "due_date": "2025-01-23",
    "created_at": "2025-01-01 00:00:00"
}
Received response:
Request failed with error:

If you're a maintainer of ths API, verify that your API is running and you've enabled CORS.
